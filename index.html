MTLabel by Tuszy

A lightweight replacement for Apple's UILabel

A simple class which renders text with CoreText framework and adds more functionality than UILabel such as custom line height or fitting the label based on text height.

Dependencies

CoreText framework

Install

1. Add CoreText framework to your project
2. Copy MTLabel header and implementation files to your project
